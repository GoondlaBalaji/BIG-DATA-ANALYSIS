# BIG DATA ANALYSIS

## Task 1 – Big Data Internship

### Tools Used
- Python
- Dask
- Pandas
- Matplotlib

### Dataset
Online Retail Dataset (UCI Machine Learning Repository)

### Objective
To perform scalable big data analysis on a large e-commerce dataset using Dask.

### Analysis Performed
1. Data Cleaning
2. Total Revenue Calculation
3. Top Countries by Revenue
4. Most Sold Products
5. Monthly Sales Trend
6. Data Visualization

### Key Insights
- United Kingdom generates the highest revenue.
- Few products dominate overall sales.
- Sales show monthly fluctuations.
- Dask enables parallel processing of large datasets efficiently.
